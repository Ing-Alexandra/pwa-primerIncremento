# Academic Review PWA

## Descripción
Aplicación Web Progresiva (PWA) para optimizar el proceso de revisión por pares académica.

## Primer Incremento Implementado
Escenario: Visualización del tablero de estado de artículos.

Funcionalidades:
- Visualizar artículos
- Cambiar estado manualmente
- Persistencia con LocalStorage
- Funcionamiento offline básico

## Arquitectura
Arquitectura en capas:
- UI (ArticleBoard)
- Lógica (ArticleManager)
- Persistencia (ArticleStorage)

## Tecnologías
- HTML5
- CSS3
- JavaScript ES6
- LocalStorage
- Service Worker

## Cómo ejecutar
1. Clonar repositorio
2. Abrir con Live Server (VSCode recomendado)
3. Acceder a index.html

## Limitaciones
- No permite subida de PDF aún
- No incluye comentarios estructurados
- No incluye autenticación
